Creating now directories or files

A directory is you computers way of organizing files so that nothing can get lost. Creating a new directory is a good way to organize and store files you don't want to lose track of.

Using Mkdir

Using the mkdir command is the perfect way to make a new directory. mkdir stands for "Make Directory".

Touch commmand

Once you create a new directory, you're going to want to put some files in there as well. You can do this using the touch command which allows you to create new files inside of specific directories.

Viewing and changing directories

Now that you have many different directories and files set up on your comupter you need a way to view them all from the command line.

Using ls

The ls commans allows you to view all the files the specific directory that you are in. This command will also allow you to see other directories that are within your current directory.

Changing directories with cd

cd is an amazing command that allows to switch directories to anywehre in your computer. To use cd type cd in to command line followed by the location of the directory that you want to enter. If you need to go back one or multiple directoreis use ../ to do so. If you need to go back further you can continue to repeat that command until you reach the level that you want.

Viewing the directory that you are in

sometimes it can get confusing to tell which directory you are in after spending a lot of time jumping around and creating amazing new files all over your computer. The best way to check wehre you are is using the pwd command which stands fore "Print Working Directory" this will print the directory you are currently working in and will give you a relative path from the root directory to where you are now.

Extra commands that provide many benifits

Once you have the genral sense of how to use the command line there are some extra commands that will boost your comfort and organization

Clear

The clear command is an easy way to clear the screen of the command line interface if its getting very crowded with past commands. This command wont clear your command history but only create a clear view for you to focus on writing even mroe amazing commands.

Up and down arrows

Using the up and down arrows are an easy and quick way of acessing other commands that you want to use again. Without needing to copy and paste you can simply use the up arrow the acess the pervious commands that you have typed. If you go to far don't worry you can go back down using the down arrow allowing you to find the exact command that you want.

Shell Command

When you start typing a command in the command line you will notice a $ just before the beginning of your command. This is called the shell command which tells you that the computer is ready to take you command. If you dont see the shell command it usually means your computer is still carrying out another task or it is stuck in a loop.
